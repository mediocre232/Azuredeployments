Microsoft Azure Developer: Implement IaaS Solutions
This repository contains the demo scripts and resources from the Microsoft Azure Developer: Implement IaaS Solutions course on Pluralsight.

The demos included in this repository cover:

Provisioning and Configuring Azure Virtual Machines
Creating and Running Containers in Azure
Use these scripts to follow along with the course and practice implementing IaaS solutions on Microsoft Azure.
